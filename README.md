Myo Index is a lightweight educational tool for learning human muscle anatomy.  
It provides quick reference information such as origin, insertion, innervation, and action.

This project is intended for study and reference only.  
It is **not** a substitute for professional medical education, diagnosis, or clinical decision-making.

Anatomical data is compiled from open educational resources and standard anatomy references.
